## ADDED Requirements

### Requirement: 代理服务器监听指定地址
代理服务器必须监听127.0.0.1:10888接收HTTP请求。

#### Scenario: 服务器成功启动
- **WHEN** 代理服务器启动时
- **THEN** 服务器监听127.0.0.1:10888

### Requirement: 转发非目标请求
代理服务器必须将不匹配拦截条件的请求转发到原始目标地址。

#### Scenario: 非视频请求被转发
- **WHEN** 客户端通过代理请求非视频URL
- **THEN** 代理转发请求到目标服务器并返回响应给客户端
